| ------- | --------------- | -------------- |
| 1.      | Google Chrome  | 103.0.5060.114  and above |
| 2.      | Mozilla Firefox | 100.0 and above      |
| 3.      | Microsoft Edge  | 104.0.1293.47 and above  |
| 4.      | Mac Safari      | 14.1 and above           |
